Configurar Cierre de Ventas

Define una fecha y hora límite para aceptar nuevas reservaciones. Después de esa fecha el sistema bloqueará automáticamente cualquier intento de reserva.

Sin fecha de cierre configurada.

Deja vacío y guarda para reabrir las ventas.
¿Cómo funciona?
  • Al configurar una fecha de cierre, el sistema bloqueará nuevas reservaciones a partir de esa fecha y hora.
  • Las reservaciones existentes no se cancelan, solo se impiden nuevas.
  • Los clientes verán un mensaje indicando que las ventas están cerradas.
  • Puedes reabrir las ventas en cualquier momento eliminando la fecha de cierre.
  • Las reservaciones B2B también serán bloqueadas.
