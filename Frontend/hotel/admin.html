Configurar Cierre de Ventas

Define una fecha y hora límite para aceptar nuevas reservaciones.

Sin fecha configurada.

Deja vacío y guarda para reabrir las ventas.
¿Cómo funciona?
  • Al configurar una fecha de cierre, el sistema bloqueará nuevas reservaciones.
  • Las reservaciones existentes no se cancelan.
  • Los clientes verán un mensaje indicando que las ventas están cerradas.
  • Puedes reabrir las ventas eliminando la fecha de cierre.
  • Las reservaciones B2B también serán bloqueadas.
